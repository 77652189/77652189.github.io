MySchool 是一个轻量级的学校课程管理系统，提供选课、作业提交、成绩查看等核心功能，是 Canvas 的低成本替代方案。

## 背景

许多中小型学校没有预算购买 Canvas 等商业平台，MySchool 提供开源的核心功能替代。

## 功能 / Features

- 学生选课与退课
- 作业发布与提交
- 成绩管理与查询
- 多角色权限（学生、教师、管理员）

## 技术栈

Java · Spring Boot · MySQL · RESTful API
