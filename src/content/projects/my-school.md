基于 Spring Boot、Spring Data JPA 和 MySQL 开发的课程管理平台，涵盖课程管理、学员报名、训练师排班及宠物档案等核心功能。

指导教授：Mamoun Abu-Samaha

## 功能模块

- **课程管理** — 训练项目创建、排期、容量控制
- **学员报名** — 在线选课、退课、进度追踪
- **训练师排班** — 日程分配与冲突检测
- **宠物档案** — 结构化记录每只宠物的训练历史

## 权限设计

实现三种用户角色的 RBAC 权限控制（管理员 / 训练师 / 学员），保障数据安全与访问隔离。

## Tech Stack

Java · Spring Boot · Spring Data JPA · Hibernate · MySQL · Maven · JUnit · RESTful API
